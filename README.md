# SignSpeak AI - Real-Time Sign Language Translator

A real-time American Sign Language (ASL) translator powered by deep learning and computer vision.

## Features

- Real-Time Recognition using webcam
- High Accuracy deep learning model
- Hand Tracking with MediaPipe
- Sentence Builder to form words
- Text-to-Speech conversion
- Interactive Streamlit UI

## Tech Stack

- Python 3.10+
- TensorFlow/Keras
- MediaPipe
- OpenCV
- Streamlit
- NumPy & scikit-learn

## Getting Started

### Installation
```bash
git clone https://github.com/YOUR_USERNAME/SignSpeak-AI.git
cd SignSpeak-AI
pip install -r requirements.txt
```

### Usage
```bash
streamlit run app.py
```

## Performance

- Test Accuracy: 80-90%
- Real-time inference at ~30 FPS
